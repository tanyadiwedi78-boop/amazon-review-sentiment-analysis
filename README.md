# Amazon Review Sentiment Analysis
An end-to-end NLP sentiment analysis system trained on **5,68,454 Amazon food product reviews** to predict customer sentiment ( Positive / Negative )

## 🎯 Model Performance 
| Metric | Score |
|--------|--------|
|Accuracy| 91.2% |
|True Positives | 81,069|
|False Negatives | 14,844|
|False Positives | 1,558|
|False Negatives | 7,687|

## ✨ Key features
1. Large Scale NLP ON 5,68,454 reviews
2. TF-IDF Vectorization
3. 91.2% Accuracy on test data
4. Complete EDA with visualizations
5. Confusion Matrix and ROC Curve

## 🛠 Tech Stack
| Category | Tools |
|-----------|-------|
| Language | Python|
